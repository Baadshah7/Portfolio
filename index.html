Internship Project

USB Write Blocker

A hybrid hardware-software digital forensics tool to protect target media from data alteration.

🎯 Problem Statement

Connecting suspect media directly to analysis hosts modifies timestamps or writes system data, destroying the integrity and admissibility of legal evidence.

🛡️ Solution

Designed an Arduino bridge that intercepts hardware write signals, coupled with a Python mounting script for read-only filesystem attachment in Kali Linux.

⚙️ Technology Stack
Python Arduino Uno Kali Linux Digital Forensics
⭐ Key Features
  • Hardware-level write inhibition via blocking bridge
  • Automated read-only device mounting daemon
  • Integrity validation using automatic SHA-256 hashing
